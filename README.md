## Simple Presenter

This project is inteded for Raspberry Pico 2 W.

Add the files from `./receiver` to one Pico2W and the files from `sender` to the other. Make sure to enter the correct pins for the LEDs.

### Functionality

Morses a custom Morsecode File while connected.
Sends a `Spacebar` key code when a switch flip is detected.
Reconnects automatically.
